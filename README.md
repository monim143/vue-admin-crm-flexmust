## What makes Flexmust special?

Flexmust is a Responsive Flat Admin CRM and written with Vue and SASS.

## Features

- Responsive layout (desktops, tablets, mobile devices)
- Built with Bootstrap v4.3.1
- Clean and Flat design
- Vue version
- CRM Pages Included
- Contacts
- Leads
- Projects
- Quotes/Estimates
- Invoices
- Users
- Roles
- Site Info
- Currency
- Payment Method
